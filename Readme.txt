Sämtliche Beispiele wurden mit der Scratch 3.0 Beta auf https://beta.scratch.mit.edu/ erstellt.

1. Mit Mozilla Firefox, Google Chrome oder Apple Safari https://beta.scratch.mit.edu/ öffnen.
2. Im Dropdownmenu unter dem Globus oben links die gewünschte Sprache wählen. (Nach Laden der 
   Sprache resettet die Entwicklungsumgebung und sämtlicher Code ist weg!)
3. Unter Datei oben links "Hochladen von deinem Computer" wählen.
4. Eine Datei wählen.
5. Zum Starten die grüne Flagge anklicken.

===============================================================================================
Klone.sb3 - SW03: Aufgabe 3 - Listen-Operationen: Essen, Umkehren, usw.
===============================================================================================
Steuerung:
----------
Pfeiltasten: Hoppeln
J-Taste:     Salto machen und Mageninhalt drehen (Liste umkehren)
Leertaste:   Köttel machen (Element mit Index 0 aus Liste entfernen)

===============================================================================================
StackImplementation.sb3 - SW01: Aufgabe 2 - Donuts als Abstrakte Datentypen
===============================================================================================
Steuerung:
----------
Push:  Baum anklicken und Wert eingeben
Pop:   Murmeltier anklicken
Stack: Vom Stapel aus Donuts in der Mitte repräsentiert

===============================================================================================
Zwei-Nebenlaeufige-Tasks.sb3 - SW09: Aufgabe 3 - Globale Flags und Events statt FutureTasks
===============================================================================================
Zur Veranschaulichung passiert alles auf demselben Sprite.
Die Sprechblase der Scratch-Cat hält als Konsolenoutput her.
