Sämtliche Beispiele wurden mit der Scratch 3.0 Beta auf https://beta.scratch.mit.edu/ erstellt.
1. Mit Mozilla Firefox, Google Chrome oder Apple Safari https://beta.scratch.mit.edu/ öffnen.
2. Im Dropdownmenu unter dem Globus oben links die gewünschte Sprache wählen.
3. Unter Datei oben links "Hochladen von deinem Computer" wählen.
4. Eine Datei wählen.
